User Test

This is a version of a tool built for a internal project and adapted to expressly test user interactions. This is not a public working project and should not be shared in its current iteration.

The goal of this user test to explore how well the interfaces are working in their current design and work on improvements as the atlas is developed further.

Open comments are welcome, however feedback on the particulars of how well the data is presented guided by the questions of the survey is especially important.

Go here to start the user test.
